## Descripción- Trabajo Practico N°2- JAVASCRIPT

El trabajo práctico se divide en dos grandes etapas que permiten avanzar desde los fundamentos de JavaScript hasta el desarrollo de interfaces web interactivas.

## Parte 1: Fundamentos y Lógica (Ejercicios 1 al 18)
Esta etapa está orientada a consolidar las bases del lenguaje y el pensamiento lógico. Los ejercicios del 1 al 8 se resuelven en consola utilizando console.log (), template strings y la extensión Code Runner. En ellos se trabajan variables, operaciones matemáticas, estructuras de control y manipulación de arreglos.
Los ejercicios 9 al 12 incorporan interacción con el usuario mediante prompt() para capturar datos y document.write() para mostrar resultados en el navegador. Finalmente, los ejercicios 13 al 18 profundizan en la creación de funciones reutilizables, abordando problemas como duplicar arreglos, sumar rangos numéricos, contar caracteres y comparar valores.
## Parte 2: Eventos, DOM y Modularización (Ejercicios 19 al 25)
En esta etapa, se adopta un enfoque orientado al desarrollo web dinámico e interactivo. Se trabaja con HTML, CSS y JavaScript para construir interfaces visuales y responder a las acciones del usuario.
El código JavaScript debe organizarse en módulos (ejercicios.js y funciones.js), separando responsabilidades en distintos archivos y utilizando "import" y "export" para compartir funciones y variables. Además, se profundiza en la manipulación del DOM mediante eventos como "click", "input" y "change", permitiendo actualizar el contenido de la página en tiempo real, modificar estilos dinámicamente y crear páginas web más estructuradas, reutilizables y profesionales. 
Asimismo, el trabajo se promueve buenas prácticas de desarrollo, como el uso de funciones flecha, declaración de las variables "let" y "const" y la organización en módulos y el trabajo colaborativo con GitHub.
